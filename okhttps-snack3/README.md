# OkHttps Gson

#### 介绍

基于 OkHttps 与 Gson 集成扩展


##### Maven

```xml
<dependency>
     <groupId>com.ejlchina</groupId>
     <artifactId>okhttps-snack3</artifactId>
     <version>3.5.3</version>
</dependency>
```

##### Gradle

```groovy
implementation 'com.ejlchina:okhttps-snack3:3.5.3'
```
