# OkHttps Xml

#### 介绍

基于 OkHttps 与 Xml 集成扩展（仅支持 Java 8）


##### Maven

```xml
<dependency>
     <groupId>com.ejlchina</groupId>
     <artifactId>okhttps-xml</artifactId>
     <version>3.1.5</version>
</dependency>
```

##### Gradle

```groovy
implementation 'com.ejlchina:okhttps-xml:3.1.5'
```
