# OkHttps Jackson

#### 介绍

基于 OkHttps 与 Jackson 集成扩展


##### Maven

```xml
<dependency>
     <groupId>com.ejlchina</groupId>
     <artifactId>okhttps-jackson</artifactId>
     <version>2.4.4</version>
</dependency>
```

##### Gradle

```groovy
implementation 'com.ejlchina:okhttps-jackson:2.4.4'
```
