# OkHttps Stomp

#### 介绍

基于 OkHttps 的 Stomp 协议实现


##### Maven

```xml
<dependency>
     <groupId>com.ejlchina</groupId>
     <artifactId>okhttps-stomp</artifactId>
     <version>3.4.7</version>
</dependency>
```

##### Gradle

```groovy
implementation 'com.ejlchina:okhttps-stomp:3.4.7'
```
