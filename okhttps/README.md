# OkHttps

#### 介绍

基于 OkHttps 核心模块


##### Maven

```xml
<dependency>
     <groupId>com.ejlchina</groupId>
     <artifactId>okhttps</artifactId>
     <version>3.1.5</version>
</dependency>
```

##### Gradle

```groovy
implementation 'com.ejlchina:okhttps:3.1.5'
```
